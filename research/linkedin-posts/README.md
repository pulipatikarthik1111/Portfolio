# LinkedIn posts (organized by author)

This folder contains notes and permalinks to recent LinkedIn posts from outbound / cold outreach experts.  
One file per author.

Naming convention: `first-last.md`

## Collection pipeline (LinkedIn portion)

01. **Expert curation** — choose creators by practitioner signal  
→ 02. **Content discovery** — collect post URLs per creator  
→ 03. **Extraction** — manual copy/export (or tooling like PhantomBuster within ToS limits)  
→ 04. **Normalization** — store post URL, date, and summary consistently  
→ 05. **Synthesis** — tag patterns (hooks, personalization style, CTA style, objections)

For the full dashboard + tooling plan, see `research/sources.md`.
