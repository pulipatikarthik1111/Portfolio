# YouTube transcripts (organized by video)

This folder is for transcript capture and notes from YouTube videos related to cold outreach and outbound pipeline building.

- One file per video
- Include the YouTube link
- Store the transcript text under a `## Transcript` section (via API or manual copy)

## Collection pipeline (YouTube portion)

01. **Content discovery** — build a video inventory (channel → videos list)  
→ 02. **Metadata** — pull title, publish date, view counts (YouTube Data API v3)  
→ 03. **Transcript extraction** — fetch transcripts (Supadata API or alternative)  
→ 04. **Normalization** — store transcript + notes in a consistent file format  
→ 05. **Synthesis** — extract frameworks, scripts, cadences, and patterns

For the full dashboard + tooling plan, see `research/sources.md`.
